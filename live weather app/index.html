Get The Most Fun Weather App

This is a Weather app that fetches live weather data from Open Weather Map API. It pulls weather data from OpenWeatherMap API based on what user input in the country text field.

based on user's input, the application pulls country's capital from PIXABAY API, and display current forcast details returned from OpenWeatherMap API.

The projects is built using: HTML, CSS, JavaScript, and BootStrap.
